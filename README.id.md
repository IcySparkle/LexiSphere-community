# LexiSphere Community

Ini adalah repositori umpan balik publik dan rilis untuk [LexiSphere](https://github.com/IcySparkle/LexiSphere-community), ekstensi Chrome untuk penerjemahan situs secara terkontrol.

## Apa Itu LexiSphere?

LexiSphere adalah ekstensi Chrome/Edge untuk penerjemahan deterministik dan mengutamakan penyimpanan lokal pada situs web pihak ketiga. Alih-alih menggunakan terjemahan mesin otomatis, Anda membangun kamus sendiri dan menerapkannya ke situs tertentu dengan aturan pencocokan URL yang presisi.

Fitur utama:
- Penerjemahan berbasis tabel string yang deterministik (tanpa tebak-tebakan)
- Aturan prefiks URL dengan cakupan yang terperinci
- Beberapa mode tampilan: mengganti teks, memberi anotasi dengan lencana overlay, atau keduanya
- Pemindaian Capture Now secara manual untuk mengumpulkan string yang belum diterjemahkan guna membangun kamus secara bertahap
- AI Assist opsional untuk saran terjemahan (manual, tidak pernah diterapkan secara otomatis)
- Mengutamakan lokal: semua data tetap tersimpan di perangkat Anda

## Melaporkan Masalah

Menemukan bug atau memiliki permintaan fitur? Silakan [buat issue baru](https://github.com/IcySparkle/LexiSphere-community/issues/new/choose).

Sebelum membuat issue baru:
1. Cari di [issue yang sudah ada](https://github.com/IcySparkle/LexiSphere-community/issues) untuk memastikan belum pernah dilaporkan sebelumnya.
2. Gunakan template issue yang sesuai.
3. Sertakan versi browser dan versi ekstensi Anda.
