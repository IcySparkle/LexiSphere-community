# LexiSphere 커뮤니티

이 저장소는 제어형 사이트 번역 Chrome 확장 프로그램인 [LexiSphere](https://github.com/IcySparkle/LexiSphere-community)의 공개 피드백 및 릴리스 저장소입니다.

## LexiSphere란?

LexiSphere는 서드파티 웹사이트를 결정론적이고 로컬 우선 방식으로 번역하기 위한 Chrome/Edge 확장 프로그램입니다. 자동 기계 번역 대신, 자신만의 사전을 구축하고 정밀한 URL 매칭 규칙을 통해 특정 사이트에 적용합니다.

주요 기능:
- 결정론적 문자열 테이블 번역 (추측 없음)
- 세밀한 범위 지정이 가능한 URL 접두사 규칙
- 다양한 표시 모드: 텍스트 치환, 오버레이 배지 주석, 또는 두 가지 모두
- Capture Now 수동 스캔으로 미번역 문자열을 수집하여 사전을 점진적으로 구축
- 선택적 AI Assist 번역 제안 기능 (수동 전용, 자동 적용 없음)
- 로컬 우선: 모든 데이터가 사용자의 기기에 저장됩니다
