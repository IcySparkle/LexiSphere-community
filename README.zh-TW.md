# LexiSphere 社群

這是 [LexiSphere](https://github.com/IcySparkle/LexiSphere-community) 的公開意見回饋與版本發佈儲存庫。LexiSphere 是一款可控式網站翻譯 Chrome 擴充功能。

## 什麼是 LexiSphere？

LexiSphere 是一款 Chrome/Edge 擴充功能，專為第三方網站提供確定性、本機優先的翻譯。不同於自動機器翻譯，您可以建立自己的字典，並透過精確的 URL 匹配規則將其套用到特定網站。

主要功能：
- 確定性字串表翻譯（無猜測）
- 基於 URL 前綴的規則，支援細粒度範圍控制
- 多種顯示模式：替換文字、以浮動標籤註解，或兩者並用
- 手動 Capture Now 掃描，收集未翻譯的字串以漸進式建立字典
- 可選的 AI Assist 翻譯建議功能（手動操作，絕不自動套用）
- 本機優先：所有資料都保存在您的電腦上
