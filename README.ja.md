# LexiSphere コミュニティ

ここは [LexiSphere](https://github.com/IcySparkle/LexiSphere-community) の公開フィードバックおよびリリース用リポジトリです。LexiSphere は、サイト翻訳を制御可能にする Chrome 拡張機能です。

## LexiSphere とは？

LexiSphere は、サードパーティのウェブサイトを確定的かつローカル優先で翻訳するための Chrome/Edge 拡張機能です。自動機械翻訳の代わりに、独自の辞書を作成し、精密な URL マッチングルールを使って特定のサイトに適用します。

主な機能：
- 確定的な文字列テーブル翻訳（推測なし）
- きめ細かいスコープ指定が可能な URL プレフィックスルール
- 複数の表示モード：テキスト置換、オーバーレイバッジによる注釈、またはその両方
- Capture Now による手動スキャンで未翻訳文字列を収集し、辞書を段階的に構築
- AI Assist による翻訳候補の提案（手動操作のみ、自動適用なし）
- ローカル優先：すべてのデータはお使いのマシン上に保持されます
