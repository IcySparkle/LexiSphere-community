# LexiSphere 社区

这是 [LexiSphere](https://github.com/IcySparkle/LexiSphere-community) 的公开反馈与发布仓库。LexiSphere 是一款可控站点翻译 Chrome 扩展程序。

## LexiSphere 是什么？

LexiSphere 是一款用于对第三方网站进行确定性、本地优先翻译的 Chrome/Edge 扩展程序。它不依赖自动机器翻译，而是让您构建自己的词典，并通过精确的 URL 匹配规则将其应用到特定站点。

主要功能：
- 确定性字符串表翻译（无猜测）
- 支持细粒度作用域的 URL 前缀规则
- 多种显示模式：替换文本、叠加标注徽章，或两者兼用
- 通过 Capture Now 手动扫描收集未翻译字符串，逐步构建词典
- 可选的 AI Assist 翻译建议功能（仅手动触发，不会自动应用）
- 本地优先：所有数据都保留在您的设备上
